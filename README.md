# 📊 KOSIS MCP 서버

KOSIS(국가통계포털) 데이터를 Claude AI가 자동으로 **검색·분석·시각화**해주는 MCP 서버입니다.  
"청년정책 보고서 쓰고 있어, 관련 통계 찾아줘" 같은 자연어 한 마디로 통계 탐색부터 차트 생성까지 한번에 처리합니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| **의도 기반 검색** | 자연어로 연구 목적을 설명하면 관련 KOSIS 통계표를 자동 탐색 |
| **데이터 조회** | 통계표 데이터를 조회해 Claude에게 전달 |
| **Claude 자체 시각화** | Claude가 직접 차트를 생성 (꺾은선·막대·파이 등 다양한 형태) |
| **다중 대시보드** | 여러 통계표를 한꺼번에 조회해 대시보드로 시각화 |
| **카테고리 탐색** | 주제별·대상별·이슈별 KOSIS 카테고리 직접 탐색 |
| **멀티유저 지원** | 서버 1개로 여러 사용자가 각자 KOSIS 키를 URL에 붙여 사용 |

---

## 🚀 사용 방법 (3단계)

### 1단계 — KOSIS 인증키 발급

1. [kosis.kr/openapi](https://kosis.kr/openapi/) 접속 후 회원가입
2. **활용신청 → 인증키 발급** 클릭 (즉시 발급)
3. 발급된 인증키 복사

### 2단계 — Claude에 연결

Claude 앱 → **Settings → Integrations → Add custom integration**에 아래 URL 입력:

```
https://kosis-mcp-production.up.railway.app/mcp?kosis_key=발급받은_인증키
```

> **서버 홈페이지**: https://kosis-mcp-production.up.railway.app  
> 홈페이지에서 인증키를 입력하면 접속 URL이 자동 생성됩니다.

### 3단계 — Claude에게 말하기

연결 후 Claude에게 바로 자연어로 요청하세요:

```
청년정책 보고서 작성 중인데, 청년 고용·주거 관련 KOSIS 통계 찾아서 분석해줘
```

```
저소득 한부모 가정을 위한 정책 마련 중이야. 관련 통계 차트로 보여줘
```

```
인구 소멸 논문 쓰고 있어. 합계출산율과 고령화율 추이 그래프 만들어줘
```

```
최근 10년간 청년 실업률 꺾은선 그래프로 보여줘
```

## 💬 사용 예시 (실제 출력 결과)

아래는 실제 MCP 호출로 얻은 출력입니다.

---

### 예시 1 — 청년 고용·주거 통계 탐색

> **"청년정책 보고서 작성 중인데, 청년 고용·주거 관련 KOSIS 통계 찾아서 분석해줘"**

자동 감지된 의도: `청년` `고용` `주거`

| 표 ID | 통계표명 | 링크 |
|-------|---------|------|
| DT_1DA7300AS | 고용보조지표(청년층, 15~29세) | [바로가기](https://kosis.kr/statHtml/statHtml.do?orgId=101&tblId=DT_1DA7300AS) |
| DT_1LC0017 | 주택소유 여부별 인구 | [바로가기](https://kosis.kr/statHtml/statHtml.do?orgId=101&tblId=DT_1LC0017) |
| DT_1DE9053S | 성별 직업교육(훈련) 경험 유무 및 시기 | [바로가기](https://kosis.kr/statHtml/statHtml.do?orgId=101&tblId=DT_1DE9053S) |

출처: 국가데이터처 KOSIS

---

### 예시 2 — 한부모 바우처 현황 차트

> **"저소득 한부모 가정을 위한 정책 마련 중이야. 관련 통계 차트로 보여줘"**

자동 감지된 의도: `한부모` `저소득`  
조회 표: **차상위 및 한부모가족 수급자 바우처 서비스 이용 현황** (DT_452001_D001)

| 연도 | 수급자 수 | 바우처 이용자 수 | 총 이용금액 |
|------|----------|---------------|-----------|
| 2019 | 943,476명 | 39,626명 | 약 1,755억원 |
| 2020 | 981,486명 | 41,054명 | 약 2,101억원 |
| 2021 | 1,016,340명 | 43,203명 | 약 2,487억원 |
| 2022 | 1,007,698명 | 43,595명 | 약 2,838억원 |
| 2023 | 1,006,834명 | 45,197명 | 약 3,348억원 |
| **2024** | **1,010,074명** | **46,838명** | **약 3,697억원** |

→ 이용금액 2019 대비 **+111%** 증가. Claude가 막대+꺾은선 복합 차트로 자동 시각화.

출처: 국가데이터처 KOSIS 「차상위 및 한부모가족 수급자 바우처 서비스 이용 현황」

---

### 예시 3 — 합계출산율·고령화 추이 그래프

> **"인구 소멸 논문 쓰고 있어. 합계출산율과 고령화율 추이 그래프 만들어줘"**

자동 감지된 의도: `저출산` `고령화` `인구소멸`  
조회 표: **시군구/합계출산율, 모의 연령별 출산율** (DT_1B81A17)

| 연도 | 전국 합계출산율 | 서울 합계출산율 |
|------|--------------|--------------|
| 2015 | 1.239 | 1.001 |
| 2017 | 1.052 | 0.836 |
| 2019 | 0.918 | 0.722 |
| 2021 | 0.808 | 0.626 |
| 2023 | 0.720 | 0.550 |
| **2024** | **0.748** | **0.581** |

→ 2015 대비 **-43.5%** 하락. Claude가 꺾은선 그래프로 자동 시각화.

출처: 국가데이터처 KOSIS 「시군구/합계출산율, 모의 연령별 출산율」

---

### 예시 4 — 청년 실업률 꺾은선 그래프

> **"최근 10년간 청년 실업률 꺾은선 그래프로 보여줘"**

자동 감지된 의도: `청년` `고용`  
조회 표: **고용보조지표(청년층, 15~29세)** (DT_1DA7300AS)

| 연도 | 고용보조지표1(%) | 실업자(천명) |
|------|---------------|------------|
| 2016 | 11.6 | 426.2 |
| 2018 | 11.4 | 407.8 |
| 2020 | 12.6 | 370.3 |
| 2022 | 9.1 | 272.4 |
| 2024 | 8.8 | 234.6 |
| **2025** | **9.2** | **231.1** |

→ 2020년 코로나 피크(12.6%) 이후 지속 개선. Claude가 꺾은선 그래프로 자동 시각화.

출처: 국가데이터처 KOSIS 「고용보조지표(청년층, 15~29세)」

## 🗂 지원 연구·정책 분야

자연어 질의에 아래 키워드가 포함되면 관련 통계를 자동으로 탐색합니다.

**대상별**: 청년 · 아동·보육 · 청소년 · 노인·고령자 · 여성 · 장애인 · 다문화 · 한부모

**이슈별**: 저출산 · 고령화 · 인구소멸 · 1인가구 · 저소득·빈곤

**주제별**: 고용·실업 · 교육 · 주거·주택 · 소득·임금 · 복지 · 보건·의료 · 인구 · 프랜차이즈 · 소상공인

**지역통계**: 시도·시군구 지역통계 · e-지방지표

**기관별**: 국가데이터처 · 국토교통부 · 보건복지부 · 교육부 · 고용노동부 등 기관명 직접 검색

## ⚙️ MCP 도구 목록

| 도구 | 설명 |
|------|------|
| `kosis_find_by_intent` | 자연어 의도 → 관련 통계표 자동 탐색 |
| `kosis_analyze` | 통계 데이터 조회 후 Claude에게 전달 (Claude가 차트 생성) |
| `kosis_dashboard` | 여러 통계를 한꺼번에 조회 (Claude가 대시보드로 시각화) |
| `kosis_browse` | KOSIS 카테고리 트리 탐색 |
| `kosis_explain` | 통계표 조사 목적·주기·대상 범위 조회 |
